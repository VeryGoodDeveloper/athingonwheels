## Design Notes

### Colors (ATOW Brand)
- Primary: Dark blue/Navy (#1a2332)
- Secondary: Bright blue (#3b82f6)
- Accent: Purple (#8b5cf6)
- Background: Dark gradient (gray-900 → black)
- Text: White/gray-300

### Typography
- Headers: Bold, modern sans-serif
- Body: Clean, readable sans-serif
- CTAs: Bold, prominent

### Components to Build
1. **VehicleCard** - Reusable for grid/list display
2. **SearchFilters** - Sidebar filter component
3. **QuoteForm** - Multi-step form for sell page
4. **Hero** - Reusable hero section
5. **Navigation** - Header with mobile responsive menu
6. **Footer** - Site-wide footer

### Mobile Considerations
- Stack filters above results on mobile
- Single column vehicle grid on mobile
- Hamburger menu for navigation
- Touch-friendly form inputs
